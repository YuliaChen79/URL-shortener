#短網址產生器

＃＃功能描述
* 輸入任一網址即可轉換成隨機短網址
* 在瀏覽器輸入短網址可連結到原來的網址

＃＃開發工具
* Node.js v14.16.0
* Express v4.17.1
* Express-Handlebars v4.0.2
* Bootstrap v5.2.3
* MongoDB
* Mongoose v5.9.7

＃＃安裝與執行步驟
1. 請先確認已安裝Node.js 與 npm
2. 開啟終端機(Terminal)，將專案clone至本地
```
git clone https://github.com/YuliaChen79/URL-shortener.git
```
3. cd進入專案資料夾並輸入：
```
npm install
```
4. 執行nodemon app.js
```
nodemon app.js
```
5. 開啟瀏覽器進入下列網址
```
http://localhost:3000
```
6. 若要暫停運行請輸入
```
Ctrl + c
```
